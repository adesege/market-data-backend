  <p align="center">A Mini Market Data Bank to show Food Market location</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Create Google API by following the instruction [https://developers.google.com/maps/documentation/geocoding/get-api-key
](here)
